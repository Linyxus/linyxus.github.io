变量分析

基尔霍夫定律

KCL

基尔霍夫电流定律

对任意电路的任意节点，在任意时刻流入该节点的电流代数和为0。

b⎲⎳k = 1ik(t) = 0

若规定流入为正，流出即为负。

推广

广义节点（封闭面）同样成立。

KVL

对任意集总电路的任意回路，在任意时刻，沿该回路的所有支路的电压降 代数和为0。

b⎲⎳k = 1uk(t) = 0

可以从电路中任意一点的电势为一确定值理解。

2b法

假设电阻有 n 个节点， b 条支路。根据拓扑约束列出 (n − 1) 个独立电流方程， (b − n + 1) 个独立电压方程。 根据元件约束列出 (b) 个VCR方程。

共计 2b 个方程，求解 b 条支路的 2b 个未知量。

支路电流法

也即代入VCR元件约束，将电压量代换为电流量。

节点电压法

形式

⎧⎪⎪⎨⎪⎪⎩ G11u1 + G12u2 + ⋯ + G1nun = is11     G21u1 + G22u2 + ⋯ + G2nun = is22     ⋯     Gn1u1 + Gn2u2 + ⋯ + Gnnun = isnn 

其中：

  • Gii 为自电导。为连接该节点所有支路的电导之和。
  • Gij 为互电导。为 i, j 两节点间所有公共支路电导之和。 恒为负 。
  • isii 为注入该节点的电流代数和。也即注入为正，流出为负。

特殊情况

  1. 含有电压源与电阻串联的支路，转化为电流源与电阻并联。
  2. 含有电流源与电阻串联的支路，忽略电阻。
  3. 含有理想电压源的支路。设支路电流 i 为未知量，并增加一个方程。 假设理想电流源位于节点 i, j 之间，则增加的方程为
ui − uj = us
  1. 电路中含有受控源。用节点电压表示控制量。

叠加定理

叠加定理基于一个基本事实：对于一个线性网络，响应与激励存在线性关系。

假设激励为 x ， 响应为 f(x) ，则线性网络满足

f(αx1 + βx2) = αf(x1) + βf(x2)

故有叠加定理：

在线性电路中，任意元件的电流和电压可视为电路中每一个激励单独作用与电路时，在该元件产生的电压与电流的 代数和。

独立作用也即将除该电源以外的所有电源置零。

置零也即

电压源:短路
电流源:开路

Notes: 功率不可叠加。

等效分析

电阻等效

  1. 等效电阻关系
电阻串联:Req = R1 + R2 + ⋯ + Rn
电阻并联:(1)/(Req) = (1)/(R1) + (1)/(R2) + ⋯ + (1)/(Rn)
  1. 分压与分流关系

电阻串联电压与电阻成正比。

电阻并联电流与电导成正比。

电源等效

  1. 电压源串联
u = u1 + u2 + ⋯ + un

等效电压源电压为串联电压源电压代数和。

只有大小方向相同的电压源才能并联，否则会产生无限大的电流。

  1. 电流源并联
i = i1 + i2 + ⋯ + in

只有大小方向相同的电流源才能串联，否则会产生无穷大的电压。

  1. 电压源与二端网络并联

只要二端网络 N 不是与电压源大小方向不同的电压源，对外等效。

  1. 电流源与二段网络串联

只要二端网络 N 不是与电流源大小方向不同的电流源，对外等效。

  1. 电流源与电压源的等效转化
Us, Rs ⇔ (Us)/(Rs), Rs

替代定理

在有唯一解的任意线性或者非线性网络中，若某一支路的电压为 uk ，电流为 ik ， 则该支路用电压为 uk 的独立电压源或是电流为 ik 的独立电流源替代，替代后 电路其他支路的电压、电流保持不变。

  1. 被替代电路必须有唯一解
  2. 其他部分与被替代部分无耦合关系。也即受控源与控制量不可被分开。

戴维南定理

任意线性二端网络 N 都可被等效为一个独立电压源 Us 与电阻 R 的串联。

其中，电压 Us 即为网络 N 的开路电压 Uoc。

电阻 R 为该网络除源后所得网络的等效输出电阻 Req 。

含受控源时 Req 的计算方法

  1. 外加电压法 对 除源 网络外加电流。
  2. 短路电流法 短路 原来 网络的两个端钮。

最大功率传输定理

对于二端网络 N ，当负载电阻 R = Req 时，负载电阻功率达到最大值

Pmax = (u2oc)/(4Req)
